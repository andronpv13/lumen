# Отчёт: Интернет-магазин восковых свечей "Lumen"

## Технологии
- PHP 8.1+ (PDO, prepared statements, password_hash)
- MySQL 8.0+
- HTML5, CSS3 (адаптивный дизайн, CSS Grid/Flexbox)
- Vanilla JavaScript

## Возможности
- Каталог товаров с фильтрацией по категориям
- Корзина покупок (сессия)
- Оформление заказов с оплатой (имитация карты / наложенный платёж)
- Отзывы и рейтинги (средняя оценка)
- Регистрация / вход с ролями
- Админ-панель: товары, категории, заказы, пользователи, отзывы, настройки магазина
- Панель модератора: товары, заказы, отзывы
- Адаптивный дизайн (mobile-first)

## Список файлов
1. `README.md` — этот отчёт
2. `db.sql` — схема БД и стартовые данные
3. `install.php` — установщик БД
4. `config.php` — конфигурация
5. `includes/db.php` — подключение к БД
6. `includes/functions.php` — вспомогательные функции
7. `includes/header.php` — шапка
8. `includes/footer.php` — подвал
9. `index.php` — главная страница
10. `product.php` — страница товара
11. `cart.php` — корзина
12. `checkout.php` — оформление заказа
13. `auth.php` — вход / регистрация
14. `logout.php` — выход
15. `orders.php` — история заказов клиента
16. `admin.php` — панель администратора
17. `moderator.php` — панель модератора
18. `assets/style.css` — стили

## Установка
1. Распакуйте файлы в веб-каталог (например, htdocs/lumen).
2. Создайте БД `lumen_shop` и выполните `db.sql` или откройте `install.php`.
3. Отредактируйте `config.php` при необходимости.
4. Админ по умолчанию: `admin@lumen.ru` / `admin123`
5. Модератор: `moderator@lumen.ru` / `mod123`
